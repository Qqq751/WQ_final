## Overview
This project fulfills the final requirement for the Self-Learning Plan. It explores spatial web design by integrating a Three.js WebGL canvas directly into the DOM structure, moving beyond traditional 2D web constraints.

## How to Run
Because this project uses ES Modules (`<script type="module">`), it must be run on a local web server to avoid CORS (Cross-Origin Resource Sharing) security errors in the browser. 

1. Open this folder in **Visual Studio Code**.
2. Install the **Live Server** extension (if you haven't already).
3. Right-click on `index.html` and select **"Open with Live Server"**.
